\centerline{\bf Manual Notation}
\vskip 14ex

\noindent In the DL\_POLY manuals specific fonts are used to
convey specific meanings:

\begin{enumerate}
\item {\em directories} - indicates UNIX file directories
\item {\sc routines} - indicates subroutines, functions and programs
\item {\sl macros} - indicates a macro (file of UNIX commands)
\item {\bf directive} - indicates directives or keywords
\item {\tt variables} - indicates named variables and parameters
\item {FILE} - indicates filenames.
\end{enumerate}
